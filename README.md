## üíé About The Project

**LuxeStore** is more than just an e-commerce site; it's a demonstration of modern web development capabilities. Built with the robust MERN stack, it offers a seamless, premium shopping experience for high-end digital assets.

The application features a sophisticated **Glassmorphism** design language, utilizing deep indigo gradients, frosted glass effects, and fluid micro-interactions powered by Framer Motion.

| Feature | Description |

## üöÄ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Ensure you have the following installed:
*   **Node.js** (v18 or higher recommended)
*   **MongoDB** (Local instance or Atlas URI)
*   **Git**

### Installation

1.  **Clone the Repo**
    ```sh
    git clone https://github.com/ravibhinchar/luxestore_mern.git
    cd luxestore_mern
    ```

2.  **Setup Backend**
    ```sh
    cd server
    npm install
    # Seed Database (Optional but recommended)
    node seed.js
    # Start Server (Port 5000)
    npm run dev
    ```

3.  **Setup Frontend**
    Open a new terminal:
    ```sh
    cd client
    npm install
    # Start Client (Port 3000)
    npm run dev
    ```
